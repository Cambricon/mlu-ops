# 4. 总结分析

总结分析主要需要考虑以下几点：

1. 需要对功能、性能测试结果有一个总结性的一句话描述；

2. 对于功能测试中发现的问题，例如精度不达标、规模受限等问题，需要显式列出；

3. 对于性能测试中 efficiency 异常、与对标硬件 latency 或 efficiency 相比差别过大（好于 / 坏于 都要包括）等情况，给出分析解释；
